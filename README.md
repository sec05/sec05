A FEM, NLA, and HPC Developer
---------------------------

I build scientific computing software focusing on the numerical methods and the code that turns mathematics into simulations. My focus is on finite element methods, numerical linear algebra, and high-performance C++. Through all my work I emphasize correctness, integrating stability analysis, unit test verification, and reproducibility.
